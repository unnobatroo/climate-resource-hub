# About Climate Science Hub

## Mission

This resource hub is a curated collection of climate science resources, tools, and opportunities. Our mission is to make climate science accessible to everyone.

## Features

- Clean & organized resources
- Simple, fast static site
- Open source
- Easy to contribute

## Contribute

1. Submit new resources via GitHub pull request
2. Report broken links or outdated information
3. Suggest improvements
4. Share with your community

## Resource Categories

- **Education** - Courses, training, learning
- **Research** - Data, reports, organizations  
- **Data & Tools** - Datasets, APIs, tools
- **Careers** - Jobs, fellowships
- **Investment** - Funds, financing

## Technical Stack

- Framework: Jekyll
- Hosting: GitHub Pages
