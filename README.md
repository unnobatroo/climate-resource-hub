# Climate Resource Hub

Compact, searchable directory for climate work, research, education, and funding.

## Add a resource

Add a typed object to `src/data.ts`:

```ts
{
  title: "Climatebase",
  url: "https://climatebase.org/",
  topic: "Career",
  subtopic: "General",
  date: "2026-07-16",
  summary: "A concise description of what the visitor will find."
}
```

Use `General` for resources that should appear in a route's “Start here” section.
